# 校园活动管理系统交付说明

## 1. 项目概述

本项目基于 Spring Boot 3.3.12、Spring Security、MyBatis-Plus、Thymeleaf 实现，当前已完成校园活动管理系统的核心业务闭环，包括：

- 用户注册、登录与权限控制
- 后台用户管理
- 活动发布、编辑、删除、查询
- 活动报名与取消报名
- 用户个人中心
- 文件上传（头像、活动图片）
- 首页统计卡片、近 7 天趋势图、热门活动缓存

## 2. 当前已完成模块

### 2.1 用户认证与权限

- 普通用户可进行注册与登录
- 使用数据库用户信息完成身份认证
- 使用 Spring Security 控制访问权限
- `ADMIN` 与 `USER` 角色已区分

### 2.2 后台用户管理

- 管理员可查看用户分页列表
- 支持新增用户
- 支持编辑用户基本信息与角色
- 支持逻辑删除用户

### 2.3 活动管理

- 支持活动列表展示与条件查询
- 支持活动详情展示
- 管理员可新增、编辑、删除活动
- 包含时间合法性校验

### 2.4 报名管理

- 普通用户可对未开始且未满员活动报名
- 用户不可重复报名同一活动
- 活动开始后不可取消报名
- 报名人数会同步更新至活动表

### 2.5 文件上传

- 支持用户头像上传
- 支持活动图片上传
- 文件元数据写入 `sys_file`
- 本地静态资源映射已配置，可通过 `/uploads/**` 访问

### 2.6 首页统计与缓存

- 首页展示用户总数、活动总数、报名总数
- 首页展示近 7 天活动/报名趋势图
- 首页展示最新活动
- 汇总统计与热门活动已接入 Redis 缓存

## 3. 关键技术实现说明

### 3.1 安全认证

- 使用 `SecurityFilterChain` 配置登录、登出与资源访问规则
- 使用 `CustomUserDetailsService` 从数据库加载用户
- 注册时使用密码加密器处理密码

### 3.2 数据持久化

- 使用 MyBatis-Plus 完成基础 CRUD 与分页查询
- 已启用逻辑删除配置
- Mapper 扫描已由启动类统一配置

### 3.3 缓存实现

- 统计汇总缓存键：`dashboard:summary`
- 热门活动缓存键：`dashboard:hotActivities`
- 当前缓存有效期：10 分钟

### 3.4 前端页面

- 使用 Thymeleaf 模板渲染
- 使用 Bootstrap 完成基础页面布局
- 使用 Chart.js 展示首页趋势图

## 4. 运行要求

### 4.1 环境要求

- JDK 17
- Maven 3.9+
- MySQL 8.x
- Redis 6.x 或更高版本

默认配置位于 `src/main/resources/application.properties`：

- 数据库：`campus_activity_management`
- 用户名：`root`

如本地环境不同，请自行修改对应配置。

## 5. 启动方式

### 5.1 初始化数据库

执行以下 SQL 文件：

- `src/main/resources/schema.sql`
- `src/main/resources/data.sql`

### 5.2 启动项目

可使用以下命令：

```bash
mvn test
mvn spring-boot:run
```

说明：

- `mvn test` 已用于验证应用上下文可正常启动
- 正式运行建议使用 `mvn spring-boot:run`

### 5.3 访问地址

默认访问地址：

- 首页：`http://localhost:8080/`
- 登录页：`http://localhost:8080/login`
- 注册页：`http://localhost:8080/register`
- 管理端：`http://localhost:8080/admin/dashboard`
